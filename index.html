About me

Drawings

I love drawing since I was a kid and usually I draw fanarts and fancomics in manga-ispired style. But currently I also collaborate with the brand Zero To Korea by drawing pictures of Korean dishes for their Instagram page. From soups, to sweets, to street food and meat-based dishes, I love to give them a realistic aspect but still keeping my typical cartoon-esque style.

Check my drawings out!

Colorings

Another thing that I love doing is digital coloring. I colored many drawings for Zero To Korea's Instagram posts, and I'm currently working on another project that is still secret but I'll soon be able to share! It's for an Italian comics publishing house, hehe!

Check my coloring works!

Frontend

I attended all the frontend courses promoted by Shecodes: SheCodes Pro, where the final project had to be a weather app that uses an API to get real-time weather data; SheCodes Responsive, from which I was able to create this portfolio; and Shecodes React, thanks to which I built the same weather app as I did with Shecodes Pro, and a dictionary app that uses a dictionary API and an images API.
